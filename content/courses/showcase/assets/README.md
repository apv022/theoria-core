# Showcase assets

- `images/cover.svg` and `images/path.svg` are original project artwork released under CC0-1.0.
- `audio/t-rex-roar.mp3` is the [MDN CC0 audio example](https://interactive-examples.mdn.mozilla.net/media/cc0-audio/t-rex-roar.mp3).
- `video/flower.mp4` is the [MDN CC0 video example](https://interactive-examples.mdn.mozilla.net/media/cc0-videos/flower.mp4).

The media files are stored locally so the compiled showcase remains playable offline.
