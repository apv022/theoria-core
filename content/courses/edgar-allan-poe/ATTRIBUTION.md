# Image and research attribution

Edgar Allan Poe portrait, public-domain historical photograph via Wikimedia Commons

Each lesson uses a distinct remote image and includes a research source link in its notes. The
remote images are not redistributed in this source package; the reader displays them when connected
and preserves the local SVG cover offline. Verify current reuse details at each linked Wikimedia
Commons file page before commercial redistribution.
