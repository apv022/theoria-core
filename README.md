# Theoria Core

Theoria Core is a fully client-side educational PWA for browsing, learning, authoring, validating,
and compiling MCF 1.0 courses. It stores personal work and progress only in the current browser.
There are no accounts, servers, databases, secrets, or paid services.

## Development

Use Node.js 24 LTS.

```bash
npm ci
npm run dev
```

Run `npm run verify` for formatting, lint, strict types, tests, and a production build. Deployment
details and the complete static-hosting procedure will be finalized with the production build.

See [the reuse report](docs/reuse-report.md) and [current project state](PROJECT_STATE.md).
